\hypertarget{group__subsystem}{}\doxysection{Simple P\+DF Generation}
\label{group__subsystem}\index{Simple PDF Generation@{Simple PDF Generation}}
Allows for quick generation of simple P\+DF documents. This is useful for producing easily printed output from C code, where advanced formatting is not required

Note\+: All coordinates/sizes are in points (1/72 of an inch). All coordinates are based on 0,0 being the bottom left of the page. All colours are specified as a packed 32-\/bit value -\/ see \mbox{\hyperlink{pdfgen_8h_a194b1db3376a39c5c11203977277f626}{P\+D\+F\+\_\+\+R\+GB}}. Text strings are interpreted as U\+T\+F-\/8 encoded, but only a small subset of characters beyond 7-\/bit ascii are supported (see \mbox{\hyperlink{pdfgen_8h_afd2de622b15e4ca5084922b445000e72}{pdf\+\_\+add\+\_\+text}} for details).
